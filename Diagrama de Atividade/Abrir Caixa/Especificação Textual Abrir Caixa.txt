Caso de Uso: Abrir Caixa
Ator principal: Administrador/Funcionário

Pré-condições: O caixa está fechado.

Fluxo Principal (cenário de sucesso):

O caso de uso inicia quando o administrador/funcionário informa que deseja abrir o caixa.
O sistema consulta o status do caixa.
O sistema verifica que o caixa está fechado.
O sistema realiza a abertura do caixa.
Pós-condições: O caixa está aberto.

Fluxos Alternativos: 2a. O caixa já está aberto.

O sistema finaliza o caso de uso sem realizar nenhuma ação adicional.
