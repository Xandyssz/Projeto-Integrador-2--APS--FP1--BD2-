Caso de Uso: Abrir Caixa
Ator principal: Administrador/Funcionário

Pré-condições: O caixa está fechado.

Fluxo Principal (cenário de sucesso):

1. O caso de uso inicia quando o administrador/funcionário informa que deseja abrir o caixa.
2. O sistema consulta o status do caixa.
3. O sistema verifica que o caixa está fechado.
4. O sistema realiza a abertura do caixa.

Pós-condições: O caixa está aberto.

Fluxos Alternativos: 
2a. O caixa já está aberto.
