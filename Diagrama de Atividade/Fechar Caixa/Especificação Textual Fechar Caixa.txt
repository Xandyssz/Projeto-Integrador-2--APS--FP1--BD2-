Caso de Uso: Fechar Caixa
Ator principal: Administrador/Funcionário

Pré-condições: O caixa está aberto.

Fluxo Principal (cenário de sucesso):

1. O caso de uso inicia quando o administrador/funcionário informa que deseja fechar o caixa.
2. O sistema consulta o status do caixa.
3. O sistema verifica que o caixa está aberto.
4. O sistema realiza o fechamento do caixa.
5. Pós-condições: O caixa está fechado.

Fluxos Alternativos: 2a. O caixa já está fechado.
