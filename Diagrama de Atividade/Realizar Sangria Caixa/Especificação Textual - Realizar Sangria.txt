Caso de Uso: Realizar Sangria de Caixa

Ator principal: Administrador / Funcionário

Pré-condições: Caixa deve estar aberto. O ator deve ter permissão para realizar a operação de sangria.

Fluxo Principal (cenário de sucesso):

1. O caso de uso inicia quando o administrador ou funcionário acessa a função de sangria no sistema.
2. O sistema verifica se o caixa está aberto.
3. O administrador/funcionário informa o valor da sangria.
4. O sistema atualiza o saldo do caixa com a sangria.
5. O sistema emite um comprovante da operação de sangria.

Pós-condições: O saldo do caixa é atualizado e o comprovante é emitido.

Fluxo Alternativo: 
2a. Se o caixa estiver fechado, o sistema solicita a abertura do caixa antes de realizar a sangria.
