Caso de Uso: Realizar Suprimento de Caixa

Ator principal: Administrador / Funcionário

Pré-condições: Caixa deve estar aberto.

Fluxo Principal (cenário de sucesso):

1. O caso de uso inicia quando o administrador ou funcionário acessa a função de suprimento no sistema.
2. O sistema verifica se o caixa está aberto.
3. O administrador/funcionário informa o valor do suprimento.
4. O sistema atualiza o saldo do caixa com o valor informado de suprimento.
5. O sistema emite um comprovante da operação de suprimento.

Pós-condições: O saldo do caixa é atualizado e o comprovante é emitido.

Fluxo Alternativo: 
2a. O Caixa está fechado fechado, o sistema solicita a abertura do caixa antes de realizar o suprimento.
